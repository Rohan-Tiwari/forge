# Forge examples

Working recipes for common Forge workflows.

| Example | Demonstrates |
|---|---|
| [`triage-inbox/`](triage-inbox/) | File watcher daemon → auto-summarize PDFs as they arrive |
| [`daily-standup/`](daily-standup/) | Cron schedule + git history + MCP gh server |
| [`codebase-tour/`](codebase-tour/) | Interactive chat for orienting in an unfamiliar repo |

Each subfolder has its own `README.md` with setup + sample tasks.
